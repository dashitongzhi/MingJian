# MingJian Commercial Offerings

MingJian Community Edition is open source under Apache 2.0.

Future commercial offerings may include:

- MingJian Cloud, a managed hosted service
- Private enterprise deployment
- Team workspaces
- Persistent multi-tenant user management
- SSO / SAML / OIDC
- Advanced RBAC and audit logs
- Usage quotas, billing, and rate limits
- Enterprise data connectors
- Private knowledge base integration
- Custom agent workflows
- Managed model gateway
- SLA, upgrade support, and deployment support

The commercial boundary should focus on hosted operations, enterprise control,
security, compliance, collaboration, support, and proprietary connectors rather
than removing already-open-sourced core capabilities.
